# project-server-blog-app-team

Project setup:
1. npm install
2. create .env file:
```
  DB_USERNAME=
  DB_PASSWORD=
  DB_DATABASE=
  DB_CLUSTER_ID=
  JWT_SECRET=
```

Project run:
npm run dev
